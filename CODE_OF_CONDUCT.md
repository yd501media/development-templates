# Code of Conduct

## Purpose
This Code of Conduct defines expectations to keep this project welcoming and
safe for everyone involved.

## Scope
It applies to all project spaces, including issues, pull requests, reviews,
and any other communication channels related to this repository.

## Expected Behavior
- Be respectful and constructive.
- Assume positive intent and collaborate in good faith.
- Provide clear, actionable feedback.

## Unacceptable Behavior
- Harassment, discrimination, or abusive language.
- Doxxing or sharing private information without consent.
- Disruptive or hostile behavior that undermines collaboration.

## Enforcement
If behavior violates this Code of Conduct, maintainers may take appropriate
action, including warnings, temporary restrictions, or removal from the project.
